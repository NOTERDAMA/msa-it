Вход и регистрация

Стань одним из нас !

Мы создали в компании все условия для развития карьеры в наиболее интересном для каждого направлении – QA Engineer, JavaScript Developer, PHP Developer, Designer и т.д.
Мы предлагаем:
- Гибкий график работы, можно работать из дома;
- Зарплата привязана к курсу доллара;
- Обучение, участие во внешних курсах, тренингах, конференциях за счет компании;
- Есть возможность развивать свои проекты. Мы всегда поддерживаем желание делиться опытом и идеями;
- Соцпакет, включающий медстраховку, курсы английского, спортивные активности, корпоративные мероприятия, кухню в офисе;
- Офис в центре города Гомель.

Вакансии

Junior QA Engineer

Необходимы навыки:
уровень знания английского языка Pre-Intermediate и выше;
понимание основных процессов и принципов тестирования (тестовая документация, критерии начала тестирования, типы и виды тестирования);
знание общей терминологии тестирования программного обеспечения и сайтов;
понимание принципов работы различных систем и программных продуктов (архитектура клиент-сервер, web/desktop/mobile приложения).

Middle JavaScript Developer

Необходимые навыки:
Уверенное владение HTML5 и CSS3, JavaScript ES6, TypeScript;
Знание одного из Front-end фреймворков: React, Angular2+, Vue;
Опыт работы на коммерческих проектах от 2-х лет в качестве Front-end разработчика;
Английский на уровне intermediate и выше;
Умение работать в команде, отсутствие коммуникативных проблем.
Будет большим плюсом: Опыт работы с Node.Js, Express.

Junior/Middle PHP Developer

Необходимы навыки:
Знание PHP и MySQL (Рассмотрим варианты если знаете C-подобный язык и БД на основе SQL);
Знание JavaScript будет плюсом;
Уверенное владение HTML5/CSS3;
Понимание принципов ООП;
Понимание принципа MVC;
Английский на уровне intermediate и выше;
Будет большим плюсом: Знание Magento
Знание систем контроля версий (GIT)
Умение работы в Linux
Понимание базовых шаблонов проектирования

Designer (UX/UI)

Необходимые навыки :
тонкое понимание последних тенденций графического дизайна; навыки работы с программным обеспечением для проектирования и инструментами каркаса (например, таких как Figma, Adobe Illustrator, Sketch, Photoshop и так далее); английский — уровень B2+, ведь нужно много общаться с клиентом;

Back-end developer Bitrix

Необходимы навыки:
Приветствуется опыт работы с CMS: 1С-Битрикс, Joomla, ModX, Drupal, WP и др.
Хорошие знания: PHP, HTML5, CSS3;
Общие знания MYSQL;
Начальные знания: JavaScript ES6, Ajax, Jquery;
Желательно опыт адаптивной вёрстки (BOOTSTRAP не ниже версии 3);
Согласие на сверхурочную работу при необходимости.
